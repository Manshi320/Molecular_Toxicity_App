# AI-Based Molecular Toxicity Prediction

This project predicts whether a molecule is Toxic or Non-Toxic using a Random Forest Machine Learning model.

## Features
- Predicts Toxic / Non-Toxic
- Displays molecular properties
- Shows prediction confidence
- Shows class probabilities

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- RDKit
- Pandas
